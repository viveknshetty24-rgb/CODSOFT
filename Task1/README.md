# Task 1: Number Game 🎯

This is my **Task 1 submission for the CodSoft Java Internship**.

The project is a **Java console-based Number Guessing Game** where the system generates a random number between **1 and 100**, and the player has to guess it within a limited number of attempts.

## 🚀 Features
- Random number generation using `Random` class
- User input using `Scanner`
- Feedback for each guess:
  - Too High
  - Too Low
  - Correct Guess
- Maximum **10 attempts**
- Multiple rounds support
- Score tracking using **rounds won**
- Replay option (`Yes/No`)
